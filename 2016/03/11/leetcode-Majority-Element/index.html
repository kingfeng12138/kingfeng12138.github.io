leetcode--Majority Element

Question

Given an array of size n, find the majority element. The majority element is the element that appears more than ⌊ n/2 ⌋ times.
You may assume that the array is non-empty and the majority element always exist in the array.

solution

Sorting--O(n log n) time

First, sort the array. the (n/2)th number is the majority Element

遍历数组O(n)
利用数组的特点，数组中有一个数字出现的次数超过了数组长度的一半。可以在
遍历数组的时候保存两个值，一个时数组中的数字num，一个是次数times。当遍历到下一个
数字的时候，先判断当前次数是否为0，如果不为零，相同则加一，不同则减去一，
如果当前次数为0，则把遍历到的值赋给num,同时times变成1.当结束遍历，num的值
就是majorityElement。
